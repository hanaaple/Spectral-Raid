AbilitySystem — GAS-lite

UE5 Gameplay Ability System 개념을 Unity에 맞게 축소 적용한 경량 수치·이펙트 관리 시스템.
Assets/Scripts/Core/AbilitySystem/

핵심 구조

AbilitySystemComponent (MonoBehaviour)
└── Dictionary<Type, AttributeSet> — Type당 하나만 등록
AttributeSet (abstract) — 수치 컨테이너
├── CharacterAttributeSet Health, MaxHealth, Stamina, MaxStamina, Speed
└── CombatAttributeSet Damage
GameplayEffect (ScriptableObject)
├── GameplayEffectType — Instant / Duration / Infinite
├── GameplayModifier[ ] — 스탯 변경 목록
└── GameplayEffectExecution[ ] — 커스텀 로직 확장

GameplayEffect 타입

Instant
즉시 BaseValue 변경 후 종료. 체력 회복·데미지 등.
Infinite
해제 전까지 CurrentValue에 지속 반영. 장비 스탯 버프.
Duration
일정 시간 후 자동 해제. 버프·디버프. Period 설정 시 주기 실행 가능.

ModifierOperation — CurrentValue 계산 공식

수식 
CurrentValue = ((Base + ΣAddBase) × MultiplyAdditive / DivideAdditive × ΠMultiplyCompound) + ΣAddFinal
Operation계산 방식용도
AddBaseBase += mag플랫 수치 증가 (Instant / Periodic)
MultiplyAdditive×계수 누산 (1 기준)퍼센트 증폭 — 여러 소스 합산
DivideAdditive÷계수 누산 (1 기준)퍼센트 감쇄
MultiplyCompound×계수 곱연산복리 적용 배율
AddFinal최종값 += mag배율 계산 후 플랫 합산
OverrideCurrentValue = mag강제 고정값 부여

AbilitySystemComponent API

메서드용도
ApplyGameplayEffectToSelf(GE)GE SO로부터 Spec 생성 후 적용. Instant는 즉시 실행 후 Invalid Handle 반환.
RemoveActiveGameplayEffect(handle)핸들로 이펙트 제거. period == 0이면 CurrentValue 재계산.
GetAttributeCurrentValue(handle)모디파이어 적용 후 CurrentValue 반환.
SetBaseAttributeValue(handle, float)BaseValue 직접 설정 후 CurrentValue 재계산.
AddAttributeSet(set)AttributeSet 등록. 같은 타입은 하나만 허용.
📌 AttributeHandle은 static readonly로 선언해 재사용. Reflection 비용은 생성 시 1회 — 런타임 읽기/쓰기는 캐싱된 FieldInfo 사용.
